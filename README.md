# Blog App

This is repo for the Blog Author front end for the Blog App, built with React.

## Installation

```bash
# Clone the repository
git clone git@github.com:goran1010/Blog-author.git

# Navigate to project directory
cd Blog-author

# Install dependencies
npm install

# Rename the environment file
cp .env.example .env
# Edit .env file as needed
```

## Usage

```bash
# Development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```
